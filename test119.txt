1. 로그인 기능
1) client - ID/PW 서버로 보내기 : 빈칸인지 아닌지 확인 후 서버로 보냄
                               서버에 해당 ID/PW 있으면 토큰 발행

2) server - ID/PW 서버에 확인 : ID는 그대로, PW는 해시값으로 암호화해서 보냄
            매칭되는 데이터 확인 후 client
+ 유효기간 - secret key로 암호화


2. 회원가입 기능                                                                        
1) 토글 : 회원가입하기 버튼 클릭시 보여여야 하는 부분(중복확인, 안내멘트, PW 재입력, 회원가입버튼, 취소버튼)

2) 서버에 ID/PW 입력 : 확인해야할 부분 - 빈칸, 중복, 형식
- 빈칸 : 빈칸인 경우 안내멘트 색변경 / 빈칸이 아닌 경우 중복확인
- 중복 : 중복인 경우 안내멘트 색변경 / 중복이 아닌 경우 형식확인
- 형식 : 형식이 틀린 경우 안내멘트 색변경 / 형식이 맞는 경우 서버에 ID/PW 입력

3) PW 해시값으로 암호화해서 저장 : 빈칸, 형식, 일치하는지 확인
